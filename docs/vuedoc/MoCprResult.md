# mo-cpr-result 

A cpr result component. 

## props 

- `value` ***Object*** (*optional*) 

  Create two-way data bindings with the component. 

## data 

- `cprApproved` 

  The cprApproved component value.
  Used to detect changes and restore the value. 

**initial value:** `false` 

## computed properties 

- `nameId` 

  Get name `cpr-result`. 
- `showAlert` 

  Show cpr alert. 

   **dependencies:** `value`
