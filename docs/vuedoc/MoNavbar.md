# mo-navbar 

A Navbar component. 

## data 

- `user` 

  The user, isLoading component value.
  Used to detect changes and restore the value. 

**initial value:** `[object Object]` 

- `isLoading` 

**initial value:** `false` 

- `username` 

  The username component value.
  Used to define a default username. 

**initial value:** `'N/A'` 

## methods 

- `logout()` 

  Get the logout and redirect.
